### 📝To do list **기획**

- 유저가 값을 입력한다
- + 버튼을 누르면 리스트에 추가 된다
- input박스가 비어 있으면 리스트에 들어가지 않는다.
- 휴지통 버튼을 누르면 리스트 목록이 삭제 된다.
- 체크 버튼을 누르면 밑줄이 그어진다.
- 끝남 탭은 끝난 아이템만, 진행중 탭은 진행중인 아이템만 뜨도록 한다.
- 진행중 완료 창에서도 삭제를 하면 바로 적용될 수 있도록 한다.
